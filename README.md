## 📋 Project Overview
Complete network design and configuration for TUF covering Amin Campus and Saleem Campus using Cisco Packet Tracer.

---

## 🛠️ Technologies Used
- Cisco Packet Tracer
- VLANs (departmental segmentation)
- OSPF Routing Protocol
- Inter-VLAN Routing (Router-on-a-Stick)
- Centralized DHCP Server
- 802.1Q Trunking

---

## 🏫 Campus Structure

| Campus | Router | Core Switch | Departments |
|--------|--------|-------------|-------------|
| Amin Campus | Cisco 2911 | Cisco 3560 | Admin, IT, Labs, Library, Faculty, Students |
| Saleem Campus | Cisco 2911 | Cisco 3560 | Admin, IT, Labs, Library, Faculty, Students |
