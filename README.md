# TUF Campus Network Design
**BSIT 6th Semester | Computer Networks | The University of Faisalabad**

## Project Overview
Complete network design and configuration for TUF covering Amin Campus 
and Saleem Campus using Cisco Packet Tracer.

## Technologies Used
- Cisco Packet Tracer
- VLANs (departmental segmentation)
- OSPF Routing Protocol
- Inter-VLAN Routing (Router-on-a-Stick)
- Centralized DHCP Server
- 802.1Q Trunking

## Files
- `Sir ibrar.final.pkt` — Cisco Packet Tracer Project File
- `TUF_Network_Project_Report.pdf` — Complete Project Report
